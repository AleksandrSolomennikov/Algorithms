# RU

## Университетский проект "Алгоритмы 1"

    Четыре основных подраздела:
    - Алогритмы сортировки
    - Задача рюкзака
    - Хэш таблица
    - Кратчайший путь в графе

В большинтсве задач были реализованы несколько вариаций решений, это сделано для замера времени выполнения программ и определния наиболее эффективных из них.

### Алгоритмы сортировки

> Algos.java

    Реализованы такие алгоритмы как: сортировка слиянием, "пузырьком", Quick Sort.


### Задача рюкзака

> KnapsackProblem/Backtracking.java

    Был использован метод бэктрекинга и динамического программирования.


### Хэш таблица

> HashtableTP.java

    Было реализовано решение задачи p-суммы с использованием базовой функции 'hashCode()' в java


### Кратчайший путь в графе

> Graphs/Graph.java        // Запуск в Algos.java

    Имплементация алгоритма Дийкстры с использованием PriorityQueue и потоков.
